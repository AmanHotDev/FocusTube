## Project Vision

FocusTube is a distraction-free, highly customized learning management player engineered to optimize student retention. By stripping away standard platform algorithmic recommendations, public comment feeds, and sidebar distractions, it creates a bulletproof educational bubble. The system provides a unified interface for structured classroom lectures while implementing deep parental gatekeeper controls.

## Architectural Capabilities & Features

* Multi-API Key Architecture: Employs a robust 4-key pooling network that automatically switches API channels to bypass quota limits effortlessly.
* Cloud-Connected Analytics: Synchronizes student names, custom school/college profiles, chronological watch history logs, and liked question banks with a Firebase Firestore database.
* Persistent Media Controls: Embedded recovery pipelines ensure that refreshing or reloading the webpage does not wipe the active video state; the stream restores instantly.
* Laptop Keyboard Mapping: Fully synchronized hardware hotkeys allow smooth tracking (Spacebar or K to Pause, M to Mute, F to Fullscreen Lock, and Arrow Keys for instant skipping).
* Dual-Mode Asset Inversion: Feature-complete dark and light theme settings running on real-time cross-frame postMessage streams. Custom visual image assets dynamically invert contrast to match dark mode aesthetics perfectly.
* Parent Control Gateway: Restricts child access to core configuration settings—such as playback speed limiter thresholds and custom skip metric variables—via a secure, password-protected administrative overlay panel.
* Embedded Help & Feedback: An integrated student support node that lets users search local FAQ accordion stacks or transmit real-time suggestions directly to the admin's database stream.

## Technical Framework Nodes

* Frontend Interface Layer: HTML5, High-Gloss Grid CSS3, JavaScript (ES6+ Architecture)
* Cloud Database Core: Google Firebase Firestore (Realtime Synchronized Mesh)
* External Gateways: YouTube Iframe Player Embedded API Core Interface

## File Hierarchy Index

* index.html — The primary vertical navigation shell routing inbound framing layers.
* home.html — Randomized, randomized category tag feed compiling video modules across multiple whitelisted playlists.
* videoplay.html — Two-column responsive media player deck containing double-tap like engines, screen locks, and keyboard hotkeys.
* whitelist.html — Student access directory tracking whitelisted lesson modules.
* channel.html — Teacher direct mapping node structuring predefined educational profiles.
* history.html — Chronological watch history log with customized popup modals and dynamic on/pause icon toggles.
* like.html — Bookmarked study resources and question banks.
* notification.html — Courseware release updates and alert nodes.
* setting.html — Child-centric display variables, distraction-free toggles, and read-only parental lock statuses.
* help.html — Support center with searchable accordion rows and a cloud-connected message transmitter.
* admin.html — Secure administrative panel enforcing credentials verification and remote courseware deployments.
